[in]pointsMatrix specifying the points through which the hyperplane should pass. Each row is one point. This matrix must be square of dimension grid.dim.
[in]positivePointVector of length grid.dim specifying a point which lies on the positive side of the interface. This point should be within the bounds of the grid. Optional. The method for determining the normal direction to the hyperplane depends on whether this parameter is supplied; see the discussion above for more details. It is an error if this point lies on the hyperplane defined by the other points.
