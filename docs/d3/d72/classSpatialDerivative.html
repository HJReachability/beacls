Computes spatial derivative.

Parameters
[out]dst_deriv_lCell vector containing the 3 or 4 left approximations of the first derivative (each the same size as data).
[out]dst_deriv_rCell vector containing the 3 or 4 right approximations of the first derivative (each the same size as data).
[in]gridGrid structure.
[in]srcSource data array.
[in]dimWhich dimension to compute derivative on.
[in]generateAllReturn all possible third order upwind approximations. If this boolean is true, then derivL and derivR will be cell vectors containing all the approximations instead of just the ENO approximation. (optional, default = 0)
[in]loop_beginIndex of loop to begin this itteration.
[in]slice_lengthslice_length of loop to begin this itteration.
[in]num_of_slicesnumber of strides to begin this itteration.
[out]x_uvecwhere to copy xs between calculation of spatial derivative

trueOperation succeeded.
