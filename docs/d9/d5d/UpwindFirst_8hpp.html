Checks two derivative approximations for equivalence. A warning is generated if either of these conditions holds: 1) The approximation magnitude > bound and the maximum relative error > bound. 2) The approximation magnitude < bound and the maximum absolute error > bound.

Normally, the return values are ignored (the whole point is the warning checks).

Parameters
[out]relErrorsThe relative error at each point in the array where the magnitude > bound (NaN otherwise).
[out]absErrorsThe absolute error at each point in the array.
[in]approx1sAn array containing one approximation.
[in]approx2sAn array containing the other approximation.
[in]boundThe bound above which warnings are generated.
Return values
trueThere is no error.
falseThere is some error
