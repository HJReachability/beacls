Computes the optimal trajectories given the optimal value function represented by (g, data), associated time stamps tau, dynamics given in dynSys.

Parameters
[out]traj(central) gradient in a g.dim by 1 vector
[out]traj_tauleft gradient
[in]gridgrid structure
[in]dataarray of g.dim dimensions containing function values
[in]tautime stamp (must be the same length as size of last dimension of data)
[in]dynSysdynamical system object for which the optimal path is to be computed
[in]extraArgsHJIPDE_extraArgs uMode specifies whether the control u aims to minimize or maximize the value function visualize set to true to visualize results projDim set the dimensions that should be projected away when visualizing fig_filename specifies the file name for saving the visualizations
Return values
trueSucceed
falseFailed
